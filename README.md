# ZEYAD SOLIMAN — FRONTEND DEVELOPER PORTFOLIO
### Red Giant · Deep Space Aesthetic · Dark UI

A frontend developer portfolio built with a "NASA mission control meets luxury editorial" design philosophy. Pure black void, crimson red accents, monospace data panels.

---

## Tech Stack

- **Core:** HTML5, CSS3, JavaScript ES6+
- **Frameworks:** React 19, Tailwind CSS 4, Bootstrap 5
- **Libraries:** Typed.js
- **Contact:** Formspree (serverless)
- **Design:** Figma

## Key Features

- **Star Field Hero** — Canvas-generated star field with red-tinted stars and nebula glow
- **Custom Cursor** — Red dot + ring cursor with hover interactions
- **Typewriter Effect** — Typed.js cycling through roles
- **Terminal Skills Panel** — Monospace capability log aesthetic
- **Active Nav Tracking** — Highlights current section on scroll
- **Scroll Animations** — Fade-in reveal on all sections
- **Contact Form** — Formspree-powered, no backend needed
- **Resume Download** — Direct PDF download
