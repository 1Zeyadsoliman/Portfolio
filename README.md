## 🛠️ Tech Stack & Tools
Built with a focus on speed, responsiveness, and a "Cyber-Neon" visual identity:

* **Core:** HTML5, CSS3 (Custom Properties & Grid/Flexbox), JavaScript (ES6+)
* **Frameworks:** [React 19](https://react.dev/), [Bootstrap 5](https://getbootstrap.com/)
* **Build Tools:** [Vite](https://vitejs.dev/) & [Tailwind CSS 4](https://tailwindcss.com/)
* **APIs:** [Groq AI](https://groq.com/) (for Lumina Ultra 4.0), Pexels API
* **Libraries:** [Typed.js](https://github.com/mattboldt/typed.js/) (Dynamic Typing Effect)
* **Deployment:** GitHub Pages / Vercel

## ✨ Key Features
* **Interactive UI:** Dynamic typing effects and smooth-scroll navigation.
